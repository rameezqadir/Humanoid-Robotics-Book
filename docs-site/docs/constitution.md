# AI Systems in the Physical World — Embodied Intelligence

**Theme:** Physical AI & Humanoid Robotics  
**Goal:** Students apply AI knowledge to control humanoid robots in simulated and real environments.

## Quarter Overview
Students learn ROS 2, Gazebo/Unity, NVIDIA Isaac, VLA, and complete a capstone Autonomous Humanoid project.

## Modules
- Module 1: ROS 2 — Robotic Nervous System
- Module 2: Gazebo & Unity — The Digital Twin
- Module 3: NVIDIA Isaac — AI Robot Brain
- Module 4: Vision-Language-Action

## Authoring Rules
Each unit must include objectives, prerequisites, lab tests, real-robot notes, and safety/ethics. All content validated via Spec-Kit Plus specs.
